# GameBerry URL Shortner

## Usage
```
Normal link: http://gameberrylabs.com/
Short link: https://gburlshort.herokuapp.com/BJ5PrZfGQ
Info: https://gburlshort.herokuapp.com/info/BJ5PrZfGQ

Private links require a passcode which is entered while creating the link before taking you to destination
```
